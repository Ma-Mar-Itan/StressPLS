# stressPLS 0.0.0.9000

* Added model specification, perturbation generation, backend abstraction,
  robustness metrics, bootstrap diagnostics, collinearity stress testing,
  predictive validation, observed subgroup heterogeneity diagnostics,
  simulation utilities, tables, plots, sensitivity reports, and optional
  backend adapter skeletons.
* The package remains estimator-neutral: users provide backends, and examples
  use deterministic toy backends only to demonstrate the API.
